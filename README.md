# Flickers Calculator

Современный калькулятор с расширенным функционалом, конвертером единиц и настройками тем.

## Возможности

### Калькулятор
- Базовые операции (+, -, ×, ÷)
- Инженерные функции (sin, cos, tan, ln, log, √)
- Константы (π, e)
- Возведение в квадрат
- Скобки

### Конвертер
- Масса (кг, г, мг, т, фунты, унции)
- Скорость (м/с, км/ч, миль/ч, узлы, фут/с)
- Давление (Па, кПа, МПа, бар, атм, мм рт.ст., psi)
- Температура (°C, °F, K)
- Длина (м, см, мм, км, мили, ярды, футы, дюймы)
- Объём (л, мл, м³, галлоны, кварты, пинты)
- Валюта (USD, EUR, RUB, GBP, JPY, CNY, CHF, CAD, AUD)
- Скидка (расчёт цены со скидкой)
- ИМТ (индекс массы тела)
- Система счисления (DEC, BIN, OCT, HEX)
- Время (секунды, минуты, часы, дни, недели, месяцы, годы)
- Площадь (м², см², мм², км², га, ар, акр, фут², дюйм²)

### Настройки
- 5 цветовых тем (фиолетовая, синяя, зелёная, красная, жёлтая)
- История вычислений (последние 50 операций)
- Сохранение настроек в localStorage

## Запуск как веб-приложение

Просто откройте `index.html` в браузере.

## Создание EXE файла

### Быстрый способ
Запустите `QUICK_START.bat` - скрипт автоматически установит зависимости и соберёт приложение.

### Ручная сборка
1. Установите Node.js с https://nodejs.org/
2. Откройте командную строку в папке проекта
3. Выполните команды:
```bash
npm install
npm run build
```

Готовый установщик будет в папке `dist/`

Подробная инструкция в файле `BUILD_INSTRUCTIONS.md`

## Технологии
- HTML5, CSS3, JavaScript
- Electron (для desktop версии)
